# toolbox

一个玩具工具箱(restful最佳实践, api模板)

## control

```shell
./control.sh {build|swagger|start|stop|restart|status}
```
- build: 构建服务编译文件
- swagger： 生成swagger文档
- start： 启动服务
- stop: 停止服务
- restart： 重启服务
- status： 服务状态
